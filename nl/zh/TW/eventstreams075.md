# 使用 MQ Light API 
{: #mql_using}

** MQ Light API 只提供於標準方案中。**
<br/>

提供 {{site.data.keyword.mql}} API 的原因是為了與較早期 {{site.data.keyword.mql}} 服務的舊版相容性。API 針對 Java、Node.js、Python 及 Ruby 提供了一個以 AMQP 為基礎的傳訊介面。
