# MQ Light API 사용 
{: #mql_using}

** MQ Light API는 표준 플랜의 일부로만 사용 가능합니다.**
<br/>

{{site.data.keyword.mql}} API는 초기 {{site.data.keyword.mql}} 서비스와의 역호환성을 위해 제공됩니다. API는 AMQP 기반 메시징 인터페이스를 Java&trade;, Node.js, Python 및 Ruby에 제공합니다.
{:shortdesc}
